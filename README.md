# Kömek — React frontend

Веб-клиент для бэкенда Kömek (авторизация, заявки, специалисты, организации).

## Запуск

1. Установить зависимости:
   ```bash
   cd frontend && npm install
   ```

2. Запустить бэкенд на порту 3000 (из папки `backend`).

3. Запустить фронтенд:
   ```bash
   npm run dev
   ```
   Приложение откроется на http://localhost:3001. Запросы к API проксируются на http://localhost:3000.

## Переменные окружения

- `VITE_API_URL` — базовый URL бэкенда. В режиме dev можно не задавать (используется proxy из `vite.config.js`). Для продакшена укажите полный URL бэкенда.

## Сборка

```bash
npm run build
```

Статика будет в `dist/`. Для продакшена задайте `VITE_API_URL` перед сборкой.
